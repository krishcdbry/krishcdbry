## 📦 Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/krishcdbry/krishcdbry.git
   cd krishcdbry
   ```

2. **Open locally:**
   - Simply open `index.html` in your browser, or
   - Use a local server:
     ```bash
     python -m http.server 8000
     # or
     npx serve
     ```

3. **View in browser:**
   ```
   http://localhost:8000
   ```

## 🚀 Deployment

### GitHub Pages (Automatic)

The repository is configured with GitHub Actions for automatic deployment:

1. Push to main branch
2. GitHub Actions automatically deploys
3. Site is live at: `https://krishcdbry.github.io/krishcdbry`

### Vercel (One-Click)

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/krishcdbry/krishcdbry)

Or manually:
```bash
npm install -g vercel
vercel
```

For detailed deployment instructions, see [DEPLOYMENT.md](DEPLOYMENT.md)

## 🎨 Customization

### Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --accent-primary: #00d4ff;
    --accent-secondary: #7c3aed;
    --bg-primary: #0a0e27;
    /* ... */
}
```

### Content
- Update personal information in `index.html`
- Modify career timeline, skills, and achievements
- Replace resume PDF with your own

### Animations
- Adjust animation speeds in `script.js`
- Customize typing text array
- Modify Matrix rain effect parameters
