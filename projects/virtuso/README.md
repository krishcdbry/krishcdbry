## 📖 Overview

Virtuso is an immersive 3D event registration platform featuring a fantasy ship journey across mystical realms. Built in 2012 for college tech fests, it transforms the mundane registration process into an epic adventure.

## ✨ Features

### 🎮 3D Interactive Experience
- **Fantasy Ship** - Fully animated 3D sailing vessel
- **Dynamic Ocean** - Realistic wave animations
- **Particle Systems** - Atmospheric effects for each realm
- **Smooth Camera** - Mouse-controlled exploration

### 🌈 Five Mystical Realms
1. **🌊 Mystical Waters** - Enchanted seas of wonder
2. **🌲 Enchanted Forest** - Mystical woodlands
3. **⭐ Celestial Skies** - Realm of stars
4. **💎 Crystal Caves** - Glittering caverns
5. **🔥 Fire Mountains** - Volcanic peaks

### 📝 Event Registration
- User-friendly registration form
- College/University information
- Event preferences selection
- Real-time form validation
- Success animations

### 🎨 Visual Excellence
- Gradient-based UI
- Glassmorphism effects
- Smooth transitions
- Responsive design
- Loading animations

## 🚀 Technology Stack

- **3D Engine**: Three.js (r128)
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Rendering**: WebGL
- **Design**: Modern glassmorphism UI
- **Architecture**: Modular JavaScript

## 💡 How It Works

### Theme System
Each realm has unique:
- Color palette
- Fog settings
- Particle effects
- Camera positions

### Ship Animation
- Gentle rocking motion (simulating waves)
- Floating lanterns with point lights
- Billowing sails
- Golden bow decoration

### User Journey
1. **Loading Screen** - Epic entrance animation
2. **Exploration** - Navigate through 5 fantasy realms
3. **Registration** - Interactive form submission
4. **Event Info** - Details about tech fest

## 🎮 Controls

### Mouse
- **Drag** - Rotate camera view
- **Scroll** - Zoom in/out

### Keyboard
- **Arrow Right/Down** - Next realm
- **Arrow Left/Up** - Previous realm
- **Escape** - Close modals

### UI Navigation
- Click realm dots on the right
- "Next Realm" button
- Navigation menu buttons

## 📚 Technical Details

### Three.js Components Used
- `Scene` - 3D world container
- `PerspectiveCamera` - Viewing perspective
- `WebGLRenderer` - GPU rendering
- `Mesh` - 3D objects
- `Lights` - Ambient and directional
- `Fog` - Atmospheric depth
- `BufferGeometry` - Efficient geometry
- `PointsMaterial` - Particle systems

### Performance Optimizations
- Geometry reuse
- Efficient particle systems
- Optimized animation loops
- Proper disposal of resources
- Responsive rendering
