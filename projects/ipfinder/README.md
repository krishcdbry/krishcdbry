# 🌐 IpFinder

**Advanced IP Address Detection & Geolocation Tool**

[![Status](https://img.shields.io/badge/Status-Active-success)](https://krishcdbry.github.io/krishcdbry)

## 📖 Overview

IpFinder is a powerful web-based tool that helps users discover their real IP address along with comprehensive geolocation and network information. Built in 2013 and continuously maintained, it provides instant access to your digital footprint.

## ✨ Features

- **🎯 Real-time IP Detection** - Instantly discover your public IP address
- **📍 Geolocation Data** - City, region, country, and postal code
- **🏢 ISP Information** - Identify your Internet Service Provider
- **🗺️ Coordinates** - Precise latitude and longitude
- **🕒 Timezone Detection** - Automatic timezone identification
- **📋 One-Click Copy** - Easy clipboard integration
- **🔄 Fallback Support** - Multiple API endpoints for reliability
- **📱 Responsive Design** - Works seamlessly on all devices

## 🚀 Technology Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **APIs**: Multiple IP detection services (ipapi.co, ip-api.com, ipwhois.app)
- **Design**: Modern gradient UI with smooth animations
- **Performance**: Optimized with fallback mechanisms

## 💡 How It Works

1. **Auto-detection**: Automatically fetches your IP on page load
2. **Multi-API**: Uses multiple APIs with fallback support for reliability
3. **Data Display**: Shows comprehensive information in organized cards
4. **Interactive**: Copy IP, refresh data, and keyboard shortcuts

## 🎨 Design Philosophy

- Clean, modern interface with gradient accents
- Intuitive card-based information layout
- Smooth animations and transitions
- Mobile-first responsive design

## 🔧 Usage

Simply open the application in your browser:

```bash
# Local development
open index.html

# Or use a local server
python -m http.server 8000
# Visit: http://localhost:8000
```

## ⌨️ Keyboard Shortcuts

- `Ctrl/Cmd + C` - Copy IP address
- `F5` or `Ctrl/Cmd + R` - Refresh data
